# HOU Universe
A personal full-featured technical portfolio site, built for demonstration and presentation only.

## 📌 Overview
HOU Universe is an independently developed personal showcase platform built with modern web technologies.
Designed with a unified space-themed visual style and glassmorphism UI, this site serves as my long-term home to display all personal projects, technical notes and interactive experiments.

It supports light / dark theme auto-switching, complete Chinese & English i18n internationalization, and fully refined responsive adaptation for both desktop and mobile devices.

## 🧩 Main Modules
- Home: Starry landing page with particle background
- About: Personal introduction and project experience timeline
- Projects: Featured project cards and detailed project records
- Notes: Collection of technical articles and study notes
- Toolbox: List of commonly used development tools and resources
- Friendly Links: Link exchange section
- Guestbook: Visitor message wall
- Contact: Contact information panel
- Playground: Interactive lightweight algorithm mini-game lab

## 🛠 Tech Stack
- Framework: Next.js 15 (App Router)
- Language: TypeScript
- Styling: Tailwind CSS
- Animation: Framer Motion
- UI Components: shadcn/ui
- Deployment: Static Export, compatible with Netlify / Vercel
